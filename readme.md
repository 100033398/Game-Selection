# Project Three: Game Selection
By Levi And Isaac

Game Selection is a lightweight Python script to allow users to play multiple games, such as:

	• High/Low Game
	• RPS (Rock Paper Scissors)
	• Blackjack
	• Hangman
	• Slot Machine
	• Tic-Tac-Toe
	• Dice Game
	• AI Trivia




## Installation

Requirements:

	• Python 3.9+
	• Openai
	• python-dotenv
	• OpenAI compatible API key, endpoint and model

  
Copy the .env using
```bash
cp .env.example .env
```

There should be a .env file attached, if not please contact us. In there will be a .env file that you must put in the same folder as the project.
